# Platzi fake store front-end

A single page application using Platzi fake store backend.

Using node v22, vite, React 18, wouter, Tanstack Query and Form, tailwind - flowbite (React) and axios.

## Install and run development build:

```bash
npm install
npm run dev
```

## To build production build run:

```bash
npm run build
npm run preview
```
